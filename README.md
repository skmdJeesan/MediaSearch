## Getting Started

### Prerequisites

Make sure you have the following installed on your system:
- Node.js (v14 or higher)
- npm or yarn package manager

### Installation

1. Clone the repository:
```bash
git clone https://github.com/skmdJeesan/MediaSearch.git
cd MediaSearch
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Start the development server:
```bash
npm run dev
# or
yarn dev
```

The application will be available at `http://localhost:5173/` (or another port if 5173 is in use)

### Building for Production

To create an optimized production build:
```bash
npm run build
# or
yarn build
```

The build artifacts will be stored in the `dist/` directory.

### Preview Production Build

To preview the production build locally:
```bash
npm run preview
# or
yarn preview
```

## Usage

1. **Search**: Enter a search query in the search bar to find media content
2. **Browse**: View results organized in tabs for Photos, Videos, and GIFs
3. **Save**: Click the save button on any media item to add it to your favorites
4. **Remove**: Click the remove button to delete media from your saved collection
5. **Switch Tabs**: Click on the tab headers to switch between different media types
